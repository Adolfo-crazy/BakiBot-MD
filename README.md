### `INSTALAR EN TERMUX`
```bash
ESCRIBE LOS SIGUIENTES COMANDOS UNO POR UNO:
> cd
> termux-setup-storage
> apt update 
> pkg upgrade 
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> pkg install yarn
> git clone https://github.com/Adolfo-crazy/BakiBot-MD
> cd BakiBot-MD
> yarn install 
> npm install
> npm update
> npm start
```
